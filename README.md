# Game-Like-Jam

A submission for the Game-Like Game jam: ![itch.io link](https://itch.io/jam/gamelike-jam-010)


### Ideas

Classic Minecraft
All sandbox, only base dungeons, no hunger and sprint, strange terrain generation, basic combat
Need: terrain alteration
Unique spin: Focus on block cration/factorization 

UniverseBlock
- Create a universe via factory/doodle god
- Add catalyist islands that allow progesssion
- Add Radiation posioning
- 3 processes of making new blocks (+ or -)
- Gases would be an ingredient 
- Use 'contraptions' to allow block creation (multi-block crafters). Ex. Wet concrete powder -> true concrete 
- Sound done ala Stars game
- Block placer/breaker
- move, jump, universal gravity, crouch to prevent going off ledge, place/break blocks
- falling off shouldn't be accidental
- ingredient blocks will remain
- Lucky block
- Solid, liquid, gas blocks
- Add intermediary inventory
